A quick implementation of abelian sandpiles

----------------------------------

## Installation

You can install `abeliansandpile` via [pip]:

```shell
pip install abeliansandpile
```


To install latest development version :

```shell
pip install git+https://github.com/leoguignard/abeliansandpile.git
```
